## IMMEDIATELY (This Sprint)

### 1. Plaintext secret storage on local builds
- **Category:** security
- **Impact:** High (scales with whether you enter any cloud API keys)
- **Effort:** M
- **Files:** `VoiceInk/Services/KeychainService.swift` (lines 14â€“17, 36â€“38, 68â€“69, 91â€“93, 112â€“113)
- **Fix:** On `LOCAL_BUILD`, replace raw `UserDefaults` with an encrypted store â€” either (a) a file in Application Support encrypted with a key held in the login Keychain (generic-password item, not the entitlement-gated group), or (b) `CryptoKit` symmetric encryption with a key derived once and stored in the login Keychain. At minimum, log a one-time warning and document the exposure so keys aren't entered casually. If you only ever use local STT (Parakeet + Ollama), the practical risk is the license string plus any key you happened to test with â€” still worth closing.

### 2. Reconcile the "fully-local, no cloud" promise with the shipped cloud stack
- **Category:** product integrity / privacy
- **Impact:** High (the privacy claim is the product)
- **Effort:** S (gate) to M (remove)
- **Files:** `VoiceInk/Transcription/Cloud/*` (15 files), `TranscriptionServiceRegistry.swift`, `ModelManagementView.swift`, `ProviderCloudManagementView.swift`, onboarding coordinators
- **Fix:** Decide the posture and make the code match it. Either (a) add a hard, default-on "Local only" switch that refuses to construct any `CloudTranscriptionService` / hide cloud providers from the UI, or (b) delete the `Cloud/` surface from this fork entirely. Today the promise is enforced only by user restraint.

## SOMEDAY (Strategic)

- **8. Adopt Swift 6 / strict concurrency.** With 584 `await` and 151 `@MainActor`, moving `SWIFT_VERSION` to 6 (or enabling `StrictConcurrency`) would surface data races at compile time. Do it incrementally per-target.
- **9. Centralize configuration.** 259 `UserDefaults` references and 38 `.shared` singletons argue for one typed settings layer (a single `AppDefaults`-style facade) to cut coupling and typo-prone string keys.
